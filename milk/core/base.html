Core/Base

Core/Base 组件是全部组件的基础组件。它包含了以下功能：

  • css reset: 样式重置 。
  • 一些标签的默认样式: 定义标签的默认风格以让标签显示地更和谐。
  • js core: 所有组件最底层依赖的 js 。
