Base

Base 包括了最基础的 CSS 和 JS 代码。所有的界面组件都依赖于 Base。

css reset

css reset 用于重置标签的默认样式。包括：

  • 删除了 html, body, form, object, iframe, img, pre 的默认边距
  • 删除了 html, body, 含链接的 img 的默认边框
  • 让 table 去边框
  • 让 hr 默认显示无阴影直线
  • 修正 legend, sub, sup, textarea, q 标签的兼容问题。

css reset - html5

重置了来自 HTML5 新标签的默认样式。包括：

  • 令 article, aside, details, figcaption, figure, footer, header, hgroup, nav, section 默认显示块级。
  • 令 audio, canvas, video 默认显示内联块。
  • 隐藏不含 controls 的 audio 标签

typography

定义了一些标签用于的默认样式。

js base

定义了全站公共的函数。
