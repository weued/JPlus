JS简介

JavaScript 被设计用来向 HTML 页面添加交互行为，是一种解释性脚本语言，通常被直接嵌入 HTML 页面。

JavaScript能做的事：

  • 网页中各元素的处理

JavaScript不能做的事：

  • 运行一个程序
  • 操作文件
  • 控制非网页区域的部分

JS基本知识

语句
JavaScript 语句是发给浏览器的命令，告诉浏览器要做的事情。 一句代码就是一个语句，就像文章的一句话。一个代码文件里可有很多语句。 每个语句结束跟一个";"号，就像每句话后跟句号。一个没有字的语句称为空语句。

语句块
一堆语句的集合，就是语句块，就像一段话。语句块要用{}包装。没有语句的语句块称为空的语句块。

函数
函数可理解为一些常用的语句块把它记下来，取个名字，称为函数，调用这些函数就可以完成对应功能。 可以根据不同函数参数，做类似的事。

变量
如数学的未知数，存储一些数据的东西。 变量往往有个名字，变量名可以是任何非关键词的有效字符（即不是符号的字符，包括中文）。
JavaScript 变量名称的规则：

  • 变量对大小写敏感
  • 变量必须以字母或下划线开始
使用var关键字定义变量，如：var 变量名 [ = 值];

注释
注释是不被运行的部分，用于说明代码功能，注释需要写在 //后（只有一行）或 /* */中。
JavaScript的代码注释和c，c++, java 完全一样。

关键字
语言使用的规定的词语叫关键字，就像专有词。

JS变量

变量的类型：

  • 数型： 常用的数字 1, 0 , -12323, 1.23, -1e2 都是数型 (Number)。
  • 布尔型： 不要被布尔这个东西吓到,其实可以叫它正误型,它只有对(true)和误(false)两个状态 (Boolean)。
  • 对象： 复杂的东西,先忽视 (Object)。
  • 函数： 通过关键字 function 定义函数 (Function)。
  • 字符： 文本， 如 “字符” , “ ” ,’’ (String)。
  • 未定义型： 当运行 var x; 这里x就是未定义型(undefined), 或者一个变量在没定义时,都是 undefined.这个类型只有一个值，undefined (undefined)。
  • 空值型 ： 不存储任何东西的值, 慢慢理会 ，这个类型只有一个值，null (null)。

JS语句

  • if...else

    if ( 条件) {
    语句1
    } else {
    语句2
    }
    当条件为真时执行语句1，否则，如果 else 部分存在的话，就执行语句2。 与“?:”不同的是，if 只是一条语句，不会返回数值。 条件是布尔值，必须用小括号括起来；语句1和语句2可以任何语句。大括号内如果只有一句,可以省略。还可以：
    if ( 条件) {
    语句1
    } else if( 条件 ){
    语句2
    }else {
    语句3
    }

  • for、while、do...while

    for (初始值;循环条件;改状态) {
    语句
    }

    先运行 初始, 判断状态循环条件, 执行语句,然后再改状态, 然后再判断循环条件,满足继续循环,其它退出。同样{}在只有一行可以不写 。

    while(条件){
    语句
    }

    如果满足,循环运行。

    do{
    语句
    }while(条件)

    先运行语句,条件满足继续循环

  • switch语句

    switch(变量){
    case 值:
    语句
    case 值2:
    语句
    case 值3:
    语句
    default: // 默认
    语句
    }

    如果变量等于一个值直接跳到那行,然后继续执行。如果想不继续执行,用 break;如果所有值没满足, 执行 default,没有默认的,直接退出。switch比if else if else if 的效率好。

  • break 和 continue 语句

    有两种特殊的语句可用在循环内部：break 和 continue。break 命令可以终止循环的运行，然后继续执行循环之后的代码（如果循环之后有代码的话）。 continue 命令会终止当前的循环，然后从下一个值继续运行。

  • For...In 声明

    For...In 声明用于对数组或者对象的属性进行循环操作。
    for ... in 循环中的代码每执行一次，就会对数组的元素或者对象的属性进行一次操作。
    for (变量 in 对象)
    {
    在此执行代码
    }
    “变量”用来指定变量，指定的变量可以是数组元素，也可以是对象的属性。

  • try...catch 语句

    try...catch 可以测试代码中的错误。try 部分包含需要运行的代码，而 catch 部分包含错误发生时运行的代码。
    try
    {
    //在此运行代码
    }
    catch(err)
    {
    //在此处理错误
    }

  • Throw 声明

    throw 声明的作用是创建 exception（异常）。 你可以把这个声明与 try...catch 声明配合使用，以达到控制程序流并产生精确错误消息的目的。
    throw(exception)
    exception 可以是字符串、整数、逻辑值或者对象。
