学 Javascript 主要是学什么

由于其它原因，很多初学者认为学 Javascript 就是用来在网页上做效果的。 其实这是片面的理解。 Javascript 只是一个语言，虽然当初它是被设计用在网页交互的，但现在，它已经是 一个完整的计算机脚本语言。Javascript不只用于网页上。

作为一门语言，学习时必须要学 Javascript 的基础语法。此外，因为 Javascript 最主要用途是 DOM 操作 和 Ajax ， 所以，有必要额外学习 DOM 和 Ajax 的知识。不然即使学会一门语言，也无法找到使用的地方。

学习 Javascript ,所需要掌握的知识有:

  • 初级: 掌握基本的 Javascript 语法。
  • 中级: 掌握 DOM 操作 和 Ajax 操作。
  • 高级: 掌握 Javascript 语法中的一些特性和难点。
  • 骨灰级: 精通各浏览器对Javascript的兼容问题，以及常见的 Javascript 技巧。

如何开始学习

根据上文，可知，学习有三个阶段: 初级、中级、高级。

初级部分: 语法

初级部分主要是看书。最好是去书店买一本，实在不愿掏钱可以从网上下一个 pdf 版本。至于什么书，这里也不额外推荐， 自己能看懂的书就是好书。也不要指望看完哪个书就直接进入高级，知识总是慢慢积累的。 如果你认为自己初级部分已经掌握不错，可以进入 《Javascript 初级语法自测》 小测，如果不合格，重新看书。如果合格， 进入 《Javascript 中级语法自测》 继续自测，如果合格，进入高级，如果不合格，争对里面的题目一一确定为什么做错了。 全部题目都确定之后，进入中级部分。

中级部分: DOM

中级部分主要还是看书和源码。读一段优秀的代码可以受益匪浅。当然还需要自己实践。如果你认为自己中级部分已经掌握不错，可以进入 《Dom 初级自测》 小测，如果不合格，重新看书。如果合格， 进入 《Dom 中级自测》 继续自测，如果合格，进入高级，如果不合格，争对里面的题目一一确定为什么做错了。 全部题目都确定之后，进入高级部分。

高级部分: 高级语法

这部分需要刻苦钻研，难点不多，但一定要拿下。如果觉得合格，可以进入: 《Javascript 高级级语法自测》
