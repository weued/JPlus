DOM 和 HTML， Javascript 的关系

DOM 意思是 Document Object Model， 文档对象模型。 DOM 是一个数据结构(一个树)，也是一个标准(W3C 标准)。 HTML 与 XML使用 DOM 作为程序接口， Javascript 是使用这个接口的工具。 使用了 DOM 接口的 HTML 也叫 DHTML ，目前所有主流浏览器都使用了 DOM， Javascript 是目前处理 HTML DOM 的最受欢迎的工具。 一般地，学习 Javascript 不学 DOM，就意味着学了兵书不打仗，纸上谈兵。

学DOM要学什么

DOM 是文档对象基础，它定义了很多对象来处理各种数据，因此，学DOM要学这些对象。那么有哪些对象呢?下面将介绍常用对象及成员。

节点 (Node)

Node 是所有 DOM 对象的基类 (抽象基类)， 所有节点都具有这些成员。 Node 对象有这些属性：

  • nodeType: 节点类型。不同类型的节点的节点类型是一个不同的整数值(在编程语言是一个 枚举 值)。常用的节点类型应该记住。
  • nodeName: 节点名字。不同类型的节点的节点名字意义不同。
  • nodeValue: 节点值。不同类型的节点的节点值意义不同。
  • parentNode: 父节点。
  • lastChild: 最后一个子节点。
  • nextSibling: 下一个兄弟节点。
  • previousSibling: 上一个兄弟节点。
  • firstChild: 第一个子节点。
  • childNodes: 子节点集合。这是抽象的集合， 它仅仅像一个集合。
  • ownerDocument: 丛属的跟节点。
Node 对象有这些方法：
  • appendChild(childNode): 插入节点到最后一个子节点的位置。
  • insertBefore(newChildNode, oldchildNode): 在 oldchildNode 前插入 newChildNode 。
  • removeChild(childNode): 删除指定的子节点。
  • replaceChild(newChildNode, oldchildNode): 替换指定的子节点。
  • cloneNode(cloneChild): 创建当前节点的副本。
是不是有几个属性很熟悉? 不要急，下面有你更熟悉的。

元素 (Element) 节点类型为 1

元素相当于 HTML 的标签， 任何一个标签都是对应了元素。Element 有这些属性：

  • attributes: 获取属性的集合。这也是抽象的集合。
  • tagName: 获取元素的标签，获取的总是大写的字符串。
  • textContent(在 IE 下为 innerText): 获取或设置元素的内容的文本。
  • innerHTML: 获取或设置元素的内容的HTML字符串。
  • style: 获取元素的样式。
  • className: 获取或设置元素的类名。
Element 有这些方法：
  • getAttribute(name)/setAttribute(name, val)/hasAttribute(name): 获取设置属性。
  • normalize(): 将多个连续子节点中的文本合并。
  • getElementsByTagName(tagName): 根据 tagName 获取指定的子节点。
  • addEventListener(event, fn, capture)/removeEventListener(event, fn, capture): 增删事件监听器。
不同的 HTML 标签对应不同的 继承于 Element 的对象，比如 div标签对应 HTMLDivElement。

属性节点 (Attr) 节点类型为 2

元素的 getAttributeNode 方法返回的就是 Attr 节点。 一个元素可以有很多属性，每个属性都是一个 Attr 。

文本节点 (TextNode) 节点类型为 3

文本节点用于保存文本。HTML 中的文本都是一个文本节点。

注释节点 (Comment) 节点类型为 8

对应 HTML 的注释。

文档节点 (Document) 节点类型为 9

熟悉的 document 就是 Document 的实例。document 节点除了提供和 Element 相似的成员外，还提供了比如 getElementById 、 createElement 等的方法。 document.createXXX 系列是动态创建节点的关键。

下面的 HTML 代码对应的DOM结构如何?


<div id="a">o<div>
这里总的是一个 DIV Element，它有1个属性节点： id, 这个属性节点的值是 a 。它有一个子节点，这个子节点是一个文本节点，内容是 o 。

document、 document.documentElement 和 document.body 的区别

document 表示整个文档， document.documentElement表示这个文档的跟节点， 相当于 html 标签， 而 document.body 是 body 标签。
