参与

有什么事可以做?

  • 正式代码的开发：这是最主要的。您可以新建一个代码文件或编辑已有的代码。具体说明见 《正式开发》
  • 代码的测试：代码测试也是要写代码的。测试的重要性仅次于开发。具体说明见 《测试》
  • UI 元素的设计：界面设计、主题制作，作为一个 UI 库是不可少的。具体说明见 《用户界面设计》
  • 文档的书写：如果您主要是为了学习而加入，那么参加文档的书写是不错的。具体说明见 《文档注释》
  • 教程的书写、整理：整理教程，表面是教人，其实是自己的复习。
  • 社区：项目的后勤。
  • 推广：项目的后勤。
这里只是列举了可以做的事，没有要求你必须选择一件事。你可以随心所欲，做自己喜欢做的事。

作个记号

当您修改了一个文件时，可以在顶部注释标明自己的名字，格式为： @时间 内容：作者 。比如：
@2011-8-13 补充文档： weued
