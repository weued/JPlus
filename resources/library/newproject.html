新建项目

要加入J+，就需要先在J+中创建一个项目。本文将指导您如何新建项目。

第一步、取一个英文项目名(可以使用数字、下划线，但不能有空格和其它特殊符号)

本页面假设项目名为weued 。

第二步、复制 assets/tools/tpl 文件夹 到 src/weued/ 文件夹.

在 src/weued/project.js 定义项目的导航页。其内容比如： 

var menus = {
				
	'菜单类目1': {
		'子菜单1': '点击子菜单1后到的链接',
		'子菜单2': '点击子菜单2后到的链接'	
	},
	
	'菜单类目2': {
		'子菜单3': '点击子菜单3后到的链接'
	}	
			
};

第三步、复制 src/weued/_tools/tpl 文件夹 到 src/weued/[分类1] 文件夹。

一个项目允许有多个分类的代码。

第四步、使用 src/weued/_tools/new.bat 为您的项目创建第一个文件。

一个分类允许有多个源码。执行 new.bat 后，输入 分类/文件名 即可在指定的分类创建新的源码。一个源码创建之后，包含一个 html 测试文件和相关的 css，js 文件。

第五步、在 assets/scripts/project.js 找到 navs 对象，添加新的导航菜单。

到此，项目已初始化完成。
